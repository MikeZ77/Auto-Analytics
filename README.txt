Setup your ssh key:

https://help.github.com/en/articles/generating-a-new-ssh-key-and-adding-it-to-the-ssh-agent

Clone the Repo:

clone git@github.com:MikeZ77/Auto-Analytics.git

Pushing a file/change to remote:

git add <file(s)>
git commit -m "some message"
git push

Moving to a branch:

git checkout <branch>

List branches:

git branch

Merging branches (for example if you wanted to merge a branch into master
keeping the master history):

git merge <branch name> master

Basic guide:

https://rogerdudler.github.io/git-guide/
